# Assets

This folder holds the images referenced by the main `README.md`.

## Current contents (placeholders)

`chat-tab.svg`, `document-scanner.svg`, and `analytics-tab.svg` are lightweight,
on-brand **mockups** of each tab, not real screenshots. They exist so the README
looks complete out of the box.

## Replacing them with real screenshots

When you run the app (`python app.py`), capture each tab and save it here. You can
either:

- Overwrite the SVGs with real PNGs and update the image links in `README.md`
  (e.g. `assets/chat-tab.png`), or
- Keep the same `.svg` filenames if you export your captures as SVG.

Suggested captures:
- `chat-tab` - the chat interface answering a sample question with the
  copy-ready summary box.
- `document-scanner` - a blank DDQ being auto-answered with the results summary.
- `analytics-tab` - the usage / cost-savings analytics view.
